# betterwheel.github.io

Landing site for **BetterWheel** and the host for its desktop-app release artifacts.

- **Site:** <https://betterwheel.github.io/> — GitHub Pages, served from the root `index.html`.
- **App source code:** <https://github.com/betterwheel/betterwheel> (Rust TUI + Tauri desktop).
- **Releases / auto-update feed:** this repo's
  [Releases](https://github.com/betterwheel/betterwheel.github.io/releases). The desktop
  updater fetches `latest.json` here anonymously, so this repo **must stay public**.

This repo is intentionally just the static site + release hosting — no application code.
